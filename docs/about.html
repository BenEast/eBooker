Ebooker is a command-line tool which lets you create ebooks and other text files from command line with ease. You don't have to be a programming expert or a nerd to use this. Anyone with a basic knowledge in computers can use this tool very easily.


Note : You must have python(2.7.xx) installed in your system to use this tool. Download and install python from here


  • Download the Ebooker zip file from Downloads
  • Extract the zip
  • Open the terminal in case of Mac and Linux. Open CMD in Windows
  • Navigate to the Ebooker folder
  • Run "python cmd-mac-*nix.py" incase of Mac and Linux. Run "python cmd-windows.py" in Windows
  • Ebooker command-line tool will open up


Ebooker commands
  • "help" : Shows list of commands
  • "exit" : Quit the session
  • "about": Read about this tool
  • "edit" : Edit/Create a file
  • "clear": Clear the screen
  • "debug": Gives you list of commonly occuring issues
