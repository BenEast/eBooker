What is eBooker?

eBooker is a command-line tool which lets you create ebooks and other text files from command line with ease. You don't have to be a programming expert or a nerd to use this. Anyone with a basic knowledge in computers can use this tool very easily. We provide just one script to run, so run it! So far, it can execute simple commands and also create and edit a file.
Note: You must have Python 3 installed in your system to use this tool. Download and install Python 3 from here by scrolling to the bottom of the page and clicking the link for your operating system.

Installation

  • Download the eBooker zip file from Downloads.
  • Extract the zip.
  • Open the Terminal for Mac and Linux, or the Command Prompt for Windows.
  • Navigate to the eBooker folder
  • Run "python3 ebooker.py".
  • The eBooker command-line tool will open up!

Commands

  • "help" - show a list of commands
  • "exit" - quit the session
  • "about" - read about this tool
  • "edit" - edit/create a file
  • "clear" - clear the screen
  • "debug" - give you a list of commonly occuring issues
